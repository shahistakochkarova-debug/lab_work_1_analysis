# Лабораторная работа №1

## Что сделано
Анализ данных о продажах. Файл data.xlsx содержит 502 продажи.

## Какие задания выполнены
1. Определила типы шкал (номинальная, порядковая, интервальная, отношений)
2. Нашла пропуски в данных (возраст, скидка)
3. Создала индикатор пропуска
4. Нашла выбросы (цена -500, возраст 150, количество 100)
5. Сравнила логарифм, корень и винзоризацию
6. Сравнила среднее и медиану (медиана лучше)
7. Показала, почему круговая диаграмма плоха
8. Сделала два графика для одних данных

## Что нашла в данных
| Проблема | Что сделала |
|----------|-------------|
| Цена -500 | Удалила строку |
| Нет возраста (19.5%) | Заполнила медианой |
| Нет скидки (10%) | Заполнила 0% |
| Возраст 150 лет | Удалила |
| Количество 100 штук | Ограничила до 10 |

## Главный вывод
Для цены лучше считать не среднее, а медиану.
Среднее = 25 321 руб
Медиана = 26 542 руб
